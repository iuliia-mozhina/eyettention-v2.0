
#############################
# differences to SWIFT 13.x #
#############################

- new implementation of alpha parameter
- flexible abort conditions for generative mode, can be defined as parameters in swiftstat.inp


#############################
# differences to SWIFT 12.x #
#############################

- tau-parameters of saccade programs have individual order parameters
	- lord: order of the labile stage
	- nord: order of the non-labile stage
	- xord: order of the execution stage
- msac- and tau-parameters are rescaled; msac = 2.0 now corresponds to 200 ms, etc.
- dropped rho parameter (no longer in use)
- pcd-parameter is gone
- reading span is inverse parabolic
- target selection is distance dependent
- ExpDat*.dat is now integrated into corpus*.dat (between word count and params of first word)
- Define model/simulation parameters in swpar.cfg (from which a temporary C file is automatically generated when calling ./build.sh)
- Define data format (number and type of columns) for corpus and fixseqin files in C-CODE/gaengine*.c
